# SCT_DS_1 - Excel Sales Dashboard

## Objective
Analyze Superstore Sales Data and create an interactive dashboard using Pivot Tables and Charts.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Dashboard Components
- Sales by Category
- Profit by Category
- Sales Trend Over Time

## Key Insights
- Technology category generated the highest sales.
- Technology category generated the highest profit.
- Office Supplies had the lowest sales.
- Furniture had the lowest profit.
